UC11: Comentar Activitat

Acceptance Criteria 11.1: Activitat invalida

Example del test d'acceptació 11.1

Posem un usuari existent ajaleo@gmail.com amb una excursio existent Delta de l'Ebre, una activitat existent Ciclisme no comprada per l'usuari previament i un text de comentari Molt divertit!, el sistema hauria de donar L'activitat encara no ha estat comprada.

Example del test d'acceptació 11.2
