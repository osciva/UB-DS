Acceptance Criteria 11.1: No hi ha text

El sistema dona opció de comentar pero el cos del text està buit

Exemple del test d'acceptació 11.1

Posem un comentari buit Delta de l'Ebre amb la excursió pagada True el sistema hauria de donar: No has escrit res

Acceptance Criteria 11.2: No has fet la excursió

El sistema dona opció de comentar pero el soci no ha fet la excursió

Exemple del test d'acceptació 11.2

Posem un comentari sense haver fet la excursió: El PedraForca "Una excursió molt maca i recomanable de fer" amb la excursió sense pagar: False el sistema hauria de donar: No es pot penjar el comentari perquè no has pagat l'excursió

Acceptance Criteria 11.3: Comentari correcte

El sistema dona opció a comentar i tots els requisits son correctes. El sistema retorna el missatge introduït
